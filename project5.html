Michael Bruno

I completed a COVID-19 data analysis project using SQL and Tableau to explore infection rates, death percentages, and vaccine rollouts globally. Using Microsoft SQL Server, I worked with real-world datasets to calculate metrics like population infection percentages, country-level death counts, and cumulative vaccination totals using advanced SQL techniques including CTEs, temp tables, and window functions. I then visualized the findings in Tableau to create an interactive dashboard for communication. This project helped solidify my skills in both data querying and insight delivery.
